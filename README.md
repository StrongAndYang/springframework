# springframework
    基于springboot2.0整合常用模块，提供整合例子避免踩坑，持续更新中。。。
    
### 项目结构
```
└── springboot
    ├── spring-cache
    │   └── spring-redis
    ├── spring-mq
    │   ├── spring-activemq
    │   └── spring-rabbitmq
    ├── spring-security
    ├── spring-security-oauth2
    ├── spring-swagger
    ├── spring-websocket
    └── sql
```
